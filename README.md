# 밸런스랩 1:1 맞춤 영양 상담 - 홍보 웹페이지

## 프로젝트 개요

밸런스랩의 실제 영양 상담 사례를 활용한 홍보 및 교육용 웹페이지입니다. 고객과의 진정성 있는 상담 과정을 자연스럽게 보여주어, 밸런스랩이 어떻게 고객 만족을 위해 노력하는지 전달합니다.

## 주요 특징

### 🎨 디자인 컨셉: "따뜻한 전문성 (Warm Expertise)"

- **의료/건강 산업의 차가운 느낌 ❌**
- **실제 카카오톡 대화처럼 자연스럽고 따뜻한 UX ✅**
- **전문성과 친근함의 균형 ✅**

### 📱 페이지 구성

1. **Hero Section**: 브랜드 메시지와 미션 강조
2. **고객 만족 3원칙**: 밸런스랩의 핵심 가치
   - 개인 맞춤형 솔루션
   - 진정성 있는 소통
   - 전문적인 근거
3. **통계 섹션**: 20분/무료/1:1 상담 핵심 정보
4. **실제 상담 케이스**: 100% 실제 대화 내용 (40개 메시지)
   - 김민지님(가명) 상담 사례
   - 식습관, 생활습관, 영양제 추천
   - 갑상선/부신/인슐린 민감도 분석
5. **CTA**: 상담 예약 유도

### 🎯 주요 기능

- **채팅 UI**: 카카오톡처럼 자연스러운 대화 인터페이스
  - 고객(왼쪽): 베이지 배경, 그린 아바타
  - 밸런스랩(오른쪽): 그린 배경, 코랄 아바타
  - 중요 메시지 하이라이트: 진한 그린 배경 + 그림자
- **스크롤 애니메이션**: Intersection Observer 기반 부드러운 등장 효과
- **반응형 디자인**: 모바일 최적화

### 🎨 디자인 특징

- **타이포그래피**:
  - 한글: Pretendard (높은 가독성)
  - 영문 헤더: Newsreader (우아한 세리프)
- **컬러 팔레트**:
  - 크림/베이지 톤: 따뜻하고 자연스러운 배경
  - 그린 악센트: 건강과 성장 상징
- **애니메이션**:
  - 페이지 로드 시 순차적 fade-in
  - 스크롤 기반 요소 등장
  - 메시지별 staggered 애니메이션

## 기술 스택

- **순수 HTML/CSS/JavaScript** (외부 의존성 최소화)
- **Google Fonts**: Pretendard, Newsreader
- **Intersection Observer API**: 스크롤 애니메이션
- **CSS Grid & Flexbox**: 레이아웃
- **CSS Variables**: 일관된 디자인 시스템
